# Logos

SVG simplificados de los productos del holding, basados en los logos originales (cubos trinity dorados, D con flecha, D con moneda feliz, T con camión).

Los SVG actuales son **aproximaciones de marca** con las paletas correctas. Para reemplazarlos por los archivos originales, simplemente sobrescribí los siguientes con los mismos nombres:

- `maclorian-x-group.svg` — holding
- `deduth-academy.svg` — academia financiera adultos
- `deduth-kids.svg` — academia financiera niños
- `traslapp.svg` — transporte materiales construcción

Si los originales son PNG o JPG, podés usar esos formatos también — solo cambiá la extensión en `index.html` donde se referencian.
